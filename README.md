# AI Adoption Barriers Framework

A diagnostic toolkit for understanding why professionals aren't adopting AI tools, and how to design targeted interventions that work.

## Overview

This framework provides:

- **6 Funnel Stages** - Identify where users drop off (Awareness → Permission → Access → Capability → Fit → Incentives)
- **9 Barrier Personas** - Understand different user archetypes and their specific blockers
- **13 Specific Barriers** - Detailed taxonomy with targeted interventions for each

## View the Framework

Open `index.html` in your browser or visit the [live site](https://brookrussi.github.io/AI_Adoption_Barriers/).

## Based On

Research into enterprise AI adoption patterns and common barriers to technology adoption in professional services.
